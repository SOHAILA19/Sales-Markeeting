# Sales & Marketing Power BI Dashboard

This project includes a **two-page interactive Power BI dashboard** designed to analyze Sales & Marketing performance across multiple years.

## Key Features

- Two-page dashboard covering **Sales KPIs** (Revenue, Profit, Cost) and **Product Performance**.
- **Data Modeling (Star Schema)** with relationships between Fact and Dimension tables.
- Custom **DAX measures** and a **DAX Date table** for time-intelligence and year comparison.
- Clear and interactive visualizations showing performance by country, sales channel, lead source, and product details.

## Tools Used

- Power BI  
- DAX  
- Power Query
